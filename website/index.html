Features

为发烧友打磨的每一处细节

从底层网络内核到界面动效，专业、稳定、现代。

双协议无感接管

后台同时运行 SOCKS5 与 HTTP/HTTPS 转发，自动接管系统代理，兼容 Steam、IDM、浏览器。

L3 物理层网卡绑定

IP_UNICAST_IF 接口强绑定，把每条连接物理钉死在指定网卡，根治同网段错网卡问题。

实时遥测大屏

基于内核计数器逐秒采样，合并速度、实时波形、各网卡负载分布一目了然。

智能调度引擎

三种调度策略——轮询、最少连接、按实时速度加权，让更快的网卡承担更多流量，聚合更高效。

链路体检与测速

一键探测各网卡出口延迟，并支持下载测速跑分，帮你挑出最健康、最快的线路参与聚合。

实时连接监控

实时连接列表展示每条连接的目标地址与所分配的出口网卡，分流全过程透明可见。

全生命周期代理保护

停止、失败、关窗、退出等所有路径都强制还原系统代理，杜绝代理残留断网。

自启 · 热键 · 通知

开机自启、启动即最小化托盘、开机自动加速；全局热键一键加速 / 停止，启停弹出系统通知。

现代界面 · 中英双语

克制的工业级控制台设计，深浅双主题，完整简体中文 / English 双语。

How It Works

物理多通道并进

加速时程序向系统注册表写入全覆盖代理链条；分流引擎收到连接后，先用 IP_UNICAST_IF 锁死出口物理网卡，再绑定源地址，强制流量剥离默认网关。

多线程下载的连接被 Round-Robin 均匀分发到每张选中的网卡，实现真正的物理带宽叠加。
